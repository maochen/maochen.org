  • Sentiment Classification System for Twitter

    Supervised machine learning classification system.
    Classify tweets extracted from twitter.com into 3 groups (Positive, Negative or Neutral).
    Maximum entropy classifier for prediction.
    Revised on Dec 2012, Backoff model added.
    Programming Language: Java SE 1.6.0
    F-Score: ~88%.
    Demo Website: TweetEmotion.com  |  Paper

  • Eucalyptus Cloud Computing Platform


    Researched cloud computing technique to establish os instance via virtual platform with IBM researchers.
    Established comparison instance, typical deployment method, deployed on several physical machines.
    Deployed instance into 2 Ubuntu Servers, utilize nested virtualization technology for both node controller and CLC, SC, Walrus.
    System Environment: Running on AMD x64 motherboard with Ubuntu's KVM + QEMU.
